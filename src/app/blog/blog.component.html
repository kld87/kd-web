Remote Life
2020-04-29

I've been lucky enough to keep working remotely during the pandemic. I've started creating and administering technical interviews at my current job with Scalable Path. I really enjoy doing it because it exposes me to a lot of different technologies, and watching candidates do coding exercises always teaches me something new. I think looking at other people's code (and watching them code) is a great way to stumble across "oh, I didn't know you could do that!" moments that make one a better developer.

I've also continued to write on Medium, although admittedly I'm not posting as often as I initially was. I still enjoy doing it but am trying to focus on producing quality articles vs. a quantity of them - although the prevailing wisdom on Medium seems to be "post often". I'm not really on the site to make money - rather get writing practice and build a bit of a portfolio, so that's fine by me. I'm lucky in that I've gotten my pieces accepted into several publications like Better Programming, gitconnected, UX Collective, and others.

I also completed levels 1-4 of Google's "foobar" challenge, which let me dust off some of my algorithm knowledge, and served as a good driver to improve my Python skills. It's definitely a powerful language, and I see why it's gaining so much popularity.

That's all for now!

Inaugural Post - New Website!
2019-10-10

I finally got around to updating my website. Although I kinda liked my old, minimal one, it was due for a refresh and a more professional feel. My portfolio section was getting a bit cluttered as well.

This site is written with Angular & Typescript, using the Clarity Design UI framework, and you can check out the code if you're interested in the guts. It's pretty standard, no WordPress (or any CMS), just bespoke and easy to manage. I'm happy how it turned out, and quite like the UI framework. I initially wanted to include my version of this interactive logo somewhere (source available here), but it turned out to be a bit too arduous right now. It looks like the proper path would actually be to use three.js plus something like Blender to make the model. I ended up with my rainbow button-bar instead (on the about page) which I'm really happy with.

Anyway, I'm not expecting to be posting here too often. As such the layout of this section is pretty minimal. In the future I may make it more robust (like adding an index and more of a list-detail format) to keep things from getting too cluttered.

Either way, thanks for stopping by, and let me know what you think of the new site!
